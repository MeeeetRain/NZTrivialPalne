# 新西兰12天情侣旅行计划（已出票版）

更新时间：2026-06-08

## 基本设定

- 出发地：深圳
- 出行时间：2026年8月8日-19日
- 总时长：12天，含深圳出发到回到深圳；实际新西兰入境住宿 9 晚
- 人数：2人，情侣出行
- 驾驶情况：无右舵车经验，但有山路和雪地驾驶经验
- 新西兰签证：两人 Visitor Visa - General 已获批，均为多次入境，有效期覆盖本次旅行
- 已购国际机票：深圳 -> 墨尔本 -> 基督城；奥克兰 -> 墨尔本 -> 深圳
- 已购新西兰国内段：**Queenstown -> Auckland，Jetstar JQ296 2026-08-17 13:00→14:50**；LPY 订单 1359046012531198（PNR YS6QHA，含 20kg 托运），ZQY 订单 1359046012408382（PNR ZTJ4HK，含 15kg 托运）
- 预算目标：两人合计约35,000元人民币为压缩目标；更现实舒适区间为38,000-43,000元
- 推荐方式：南岛轻自驾 + 米尔福德峡湾跟团巴士 + 皇后镇公共交通 + 奥克兰回程缓冲一晚

## 每日重点

### D1-D2：深圳 -> 墨尔本 -> 基督城

- D1 8/8 晚上从深圳宝安 T3 出发，国际航班起飞时间为 23:55。
- D2 8/9 中午抵达墨尔本，停留 6小时10分；晚上飞基督城，23:35 抵达。
- 抵达基督城已经很晚，第一晚只做入境、取行李、打车或酒店接驳、睡觉。
- 特别注意：机票行程单提示 **baggage may not be checked through**，出发前必须和深圳航空、Air New Zealand 或出票平台确认两程行李是否直挂。
- 若行李不能直挂，墨尔本需要清关、取行李、重新托运，澳洲过境签就更是刚需。

### D3：基督城到 Tekapo

- 这是第一天右舵左行，建议睡够后再取车，不要一早硬撑。
- 上午：基督城机场取车，检查车况、拍视频、确认雪链和保险。
- 中午：采购水、零食、早餐、车载用品。
- 午后：基督城开往 Lake Tekapo，约 3 小时，尽量 16:30 前抵达。
- 傍晚：牧羊人教堂和湖边散步。
- 晚上：观星和温泉二选一。天气好优先观星，天气差优先温泉。

### D4：Tekapo 到 Twizel / Mount Cook

- 这一天不再直接赶到 Wanaka，改为新增一晚 Twizel / Mount Cook，降低冬季赶路风险。
- 重点是 Lake Pukaki、Peters Lookout、Aoraki/Mount Cook Visitor Centre。
- 天气好：优先 Kea Point Track 或 Hooker Valley 开放路段。
- 天气差：只做游客中心和安全观景点，不强行徒步。
- 住宿选择：预算舒适优先 Mount Cook Village；预算控制优先 Twizel。

### D5：Twizel / Mount Cook 到 Wanaka

- 上午：根据天气补一个库克山短步道或直接出发。
- 中午：经 Omarama / Lindis Pass 一线去 Wanaka。
- 下午：Wanaka 湖边散步、That Wanaka Tree。
- 雨雪天备用：Puzzling World 或镇上咖啡馆休整。
- 当晚不安排复杂项目，为第二天最后驾驶日留体力。

### D6：Wanaka 到 Queenstown（最后驾驶日，当天还车）

**关键调整**：D6 是租车最后一天，傍晚直接在 ZQN 机场还车，D7-D10 全部用公交/班车/团车。原因：皇后镇区内活动不需要自驾，公共交通、团车和雪场班车更省钱也更安全。

#### 当天节奏

- 早上：Wanaka 湖边散步 + That Wanaka Tree（约 1 小时）。
- 上午：Wanaka 出发，**经 Arrowtown 顺路停 1.5-2 小时**（午餐 + 华人淘金定居点 + 主街）。
- 下午：Arrowtown -> Queenstown，先把行李放进酒店或寄存。
- 傍晚 16:00-17:00：开到 **Queenstown Airport 还车**。
- 还车后：在机场买 Bee Card，Route 1 公交 / Uber 回镇中心酒店。
- 晚上：镇中心晚餐 + 早睡（D7 米尔福德长途）。

#### 路线选择

- Crown Range 风景更好，但冬季可能结冰或要求雪链；路况不好就走 Cromwell。
- 当天早上 7:00 看 [NZTA 路况](https://www.journeys.nzta.govt.nz/traffic) 决定。

#### 时间风险

- 如果上午 Wanaka 出发慢、Arrowtown 停太久，傍晚还车可能赶上门店关门。租车下单时必须确认 Queenstown Airport 还车门店营业时间和 after-hours return 条款。

### D7：米尔福德峡湾

- 坚持跟团，不建议自驾 Milford Road。
- 提前准备晕车药、充电宝、保暖外套、防水外层、零食和水。
- 当晚不要再安排付费项目。

### D8：皇后镇休整日 + 滑雪装备预取日（公共交通日）

D7 米尔福德很长，D8 重点是恢复体力 + **下午去 Queenstown Snow Centre 提前取 MyPass 卡和试装备**，为 D9 滑雪做准备。

#### 上午-中午：放松

- 睡到自然醒，镇中心吃早午餐（Vudu Café 推荐）+ Queenstown Gardens 湖边散步。
- 中午：镇中心吃饭，能量补足。

#### 下午 15:00-17:00：Queenstown Snow Centre 一站式办理 ⭐重要

地址：**25 Shotover Street**（步行从镇中心 5-8 分钟，主入口在 Shotover St，后门在 Duke St）  
官方网站：[nzski.com/mypass](https://www.nzski.com/mypass/)  
雪季营业时间：6 月中-10 月中，每天 7:30-19:00

**MyPass 是什么**：NZSki 的官方 RFID 滑雪卡，相当于滑雪日的"身份证"。一张卡能装下所有付款过的：缆车通票（Superpass）、装备租赁、班车票、课程预订、山上消费（可绑信用卡）。两人各一张，**丢了补卡 NZD 5**。

**当天 60-90 分钟的流程**：

| 步骤 | 内容 | 时长 |
|---|---|---|
| 1. 取 MyPass | NZSki 柜台扫预订邮件 + 出示护照，两人发两张卡 | 5-10 分钟 |
| 2. 选装备 | 转到 Snow Centre Rentals 区，选 Recreational 套装 | 1 分钟 |
| 3. **量脚 + 试雪鞋** ⭐关键 | 当场穿走 5-10 分钟，脚趾轻触+脚跟不能上下移动，不合脚换 2-3 双直到适合 | 15-20 分钟 |
| 4. 选板 + 调绑定 | 按身高+水平+体重 | 10 分钟 |
| 5. 试头盔 | 选合适尺码 | 3 分钟 |
| 6. 全部登记到 MyPass | — | 5 分钟 |
| 7. **寄存装备过夜** | 告诉店员 "I'll come back tomorrow morning"，店员贴名签放领取区 | 1 分钟 |
| 8. 确认班车 | 跟柜台确认班车票已在 MyPass 上，明早 7:30 Duke St 9 号上车 | 2 分钟 |
| 9. 拿纸质 trail map | 问店员当日 Coronet Peak 建议 | 3 分钟 |

**为什么必须提前一天去**：早上当天去要排队 30-45 分钟，且雪鞋不合脚会毁掉一整天体验。

### D9：皇后镇正式滑雪日 ⛷️

两人**初中级水平（刚完全掌握平行式）**，安排为全天 Coronet Peak。

#### 雪场首选：Coronet Peak（针对初中级，强推）

[官方网站](https://www.coronetpeak.co.nz/) · [Snow Report 雪况报告](https://www.coronetpeak.co.nz/the-mountain/snow-report/) · [Trail Map 雪道地图](https://www.coronetpeak.co.nz/the-mountain/trail-map/)

- **70% 地形适合初中级**（25% 绿 + 45% 蓝）
- 主导地形是**宽阔修压道（groomed cruisers）**，容错性强
- 高速缆车（Coronet Express、Greengates）+ Big Easy 魔毯，**一天能刷 15-20 趟**，比 Remarkables 多 30-50%
- 距皇后镇市中心 25 分钟车程（Remarkables 是 45 分钟），早上能晚起 30 分钟
- 学习曲线友好：绿 → 淡蓝 → 蓝 → 深蓝，循序渐进有缓冲

#### 雪场备选：The Remarkables（针对你们水平不强推）

[官方网站](https://www.theremarkables.co.nz/) · [Mountain Report 山况报告](https://www.theremarkables.co.nz/the-mountain/the-mountain-report/) · [Trail Map](https://www.theremarkables.co.nz/the-mountain/trail-map/)

- 风景更好（直接看 Lake Wakatipu 全景），但**雪道偏陡，蓝色线后面经常直接接黑级**
- 适合中高级和**愿意被挑战**的滑手，**初中级容易被吓退**
- 仅作为 Coronet Peak 因风/雪关闭时的备用方案——Superpass 可无缝切换，不用补差价

#### 进阶课程（可选但强推）

刚过平行式拐点，**一节专项课会让接下来 3-5 年的滑雪体验质变**——尤其学 **carving（卡宾/刻滑）入门**。

- 课程：Coronet Peak Group Lesson 90 分钟
- 单价：**NZD 99/人** × 2 = NZD 198（约 ¥790）
- 时段：10:00-11:30（热身后正合适）
- 预订：[NZSki 课程页面](https://www.nzski.com/coronet-peak/lessons/)
- 价值：一节课能省你 5-10 个自学日的弯路

#### 装备清单（自带 vs 现场租）

**必须自带**（国内出发前准备）：

- 滑雪服（外层防水透气、内层抓绒，**不要普通羽绒服**，湿透就废）
- 防水手套（带备用一双，湿了能换）
- 抗紫外线雪镜（**不要墨镜**，雪盲很严重）
- 防晒霜 SPF50+（高海拔紫外线极强）
- 保暖打底（merino 羊毛或 polypro 速干）
- 高 SPF 唇膏
- 暖宝宝 4-6 片（应急保暖）

**现场租**（Snow Centre 已含在 Recreational 套装内）：

- 雪板 / 雪杖 / 雪鞋 / 头盔
- **不建议带自己的板**，国际航班托运费很高

#### 天气差怎么办

8 月偶尔遇风太大或雪太软，雪场可能局部关闭。备用方案：

1. **8/15 晚 21:00 + 8/16 早 6:00**：查 [Coronet Peak Snow Report](https://www.coronetpeak.co.nz/the-mountain/snow-report/) 和 [The Remarkables Mountain Report](https://www.theremarkables.co.nz/the-mountain/the-mountain-report/)
2. **如果 Coronet Peak 关闭**：Superpass 直接切到 **The Remarkables**，班车上车点同样在 Duke St 9 号（告诉司机切换）
3. **两个雪场都关**：改 **Skyline Gondola 缆车**（NZD 55/人，山顶观景+午餐）+ Onsen 温泉
4. 提前把 Skyline 和 Onsen 加入心理 Plan B，不用现订也来得及

### 澳洲过境签 771 办理要点

中国大陆普通护照经墨尔本转机去新西兰，一般不能使用澳洲免签过境。即使不出机场，只要不属于澳洲 TWOV 免签过境名单，仍应提前办理澳大利亚 Transit visa subclass 771。当前机票已经出票，且去程、回程都经墨尔本，因此这是 **2026-06-08 起的最高优先级事项**。

基本信息：

- 签证类型：Transit visa subclass 771。
- 用途：经澳大利亚前往第三国，例如深圳 -> 墨尔本 -> 基督城。
- 停留上限：每次过境澳大利亚不超过72小时。
- 申请费：澳洲政府签证申请费为免费。
- 申请方式：通过澳大利亚 ImmiAccount 在线申请。
- 中国申请人：澳大利亚驻华使馆说明，771过境签申请需通过 ImmiAccount 在线递交。

建议办理顺序：

1. 新西兰 Visitor Visa 已获批，可直接用于澳洲771材料。
2. 确认航班是联程票、行李是否直挂、墨尔本中转时间是否正常。
3. 用已购完整往返机票准备澳洲771材料。
4. 在 ImmiAccount 创建或登录账号，选择 Transit visa subclass 771。
5. 两个人都要各自拿到澳洲签证批准。可以用同一个 ImmiAccount 管理申请，但每位旅客都需要自己的申请和签证结果。
6. 上传材料后提交。771本身免费，但如果被要求录指纹/拍照，VFS生物识别中心可能收服务费。
7. 如果收到 biometrics requirement letter，要按信件要求预约 VFS / Australian Biometrics Collection Centre，带护照原件和生物识别要求信去采集。
8. 收到签证批准信后，核对姓名、护照号、有效期、入境次数和允许停留时间。

建议准备材料：

- 护照信息页。
- 身份证正反面。
- 白底或浅色背景证件照。
- 完整往返机票订单，显示经墨尔本转机。
- 从澳大利亚前往第三国的确认机票，例如 MEL -> CHC。
- 新西兰签证批准信，证明可以进入目的地国家。
- 简短英文说明信，说明只是经澳洲过境去新西兰旅游，不在澳洲停留或工作。
- 在职证明/准假信、银行流水等中国约束材料，作为补充。

特别注意：

- 如果去程和回程都经墨尔本，不要默认一张771一定覆盖两次过境。最终以签证批准信上的入境次数、有效期和条件为准；如果只覆盖一次，需要再申请回程过境签，或改用其他适用澳洲签证。
- 如果航班不是联程票、需要在墨尔本取行李再托运，仍然需要澳洲签证，并且要留足中转时间。
- 机票已出票，接下来不要再拖澳洲过境签。
- 如果澳洲过境签无法及时获批，需要尽早评估改签不经澳洲的航线，避免临近出发被动。

如果出入新西兰都经墨尔本：

- 每个人都要确保澳洲签证覆盖两次过境：去程 SZX -> MEL -> CHC，以及回程 AKL -> MEL -> SZX。
- 申请 771 时上传完整往返机票和新西兰签证，在说明信里明确写出两次经墨尔本过境日期和中转时间。
- 提交后重点看批准信的 travel facility / entries / validity。如果批准信显示多次入境且有效期覆盖往返日期，通常一份批准即可覆盖两次过境。
- 如果批准信只给单次入境，不能用它覆盖回程，需要再为回程申请一次771，或改用其他允许多次入境并可过境的澳洲签证。
- 771本身免费，但两个人可能都被要求做生物识别；如果需要去 VFS，服务费按人收。
- 实操上，建议新西兰签证获批后、机票仍可退改时提交澳洲771；等两个人的澳洲签证批准信都确认覆盖往返后，再锁定不可退项目。

### 使用技巧

1. **离线下载**：iPhone Apple Maps 已支持区域离线下载（设置 → 地图 → 离线地图），出发前在 Wi-Fi 环境下载南岛 Tekapo-皇后镇 区域，进入信号差的山区也能用。
2. **CarPlay**：租车如果带 CarPlay（GO Rentals 部分车型有），可上车直接连 iPhone 用 Apple Maps 导航。
3. **实时路况**：Apple Maps 在新西兰主要公路有实时路况，雨雪封路会标黄/红，比单看 NZTA 网页直观。
4. **替代方案**：如 Apple Maps 在某些南岛偏远路段路线计算异常，备用打开 **Google Maps**（链接里把 `maps.apple.com` 换成 `www.google.com/maps`，参数 `&dirflg=d` 改成 `&travelmode=driving` 即可）。
5. **手机分享**：在 iPhone 上长按链接 → 拷贝 → AirDrop 给同行 iPhone，双人都有导航。
